# Helix — требования и статус

## Видение

Веб-приложение для ПК (десктоп-first), помогающее **музыканту** писать музыку: интерактивные инструменты с подсветкой по теории, квинто-квартовый круг с диатоникой и вторичными доминантами, поисковик аккордов с закреплением. Прототип создан через Claude Design, дальнейшее развитие — через Claude Code.

Приоритет: быстрый доступ к ключевым решениям при сочинении (тональность → ступень → аппликатура) без лишних кликов. Поддержка узких viewports (пользователь использует тайлинг).

## Дизайн-система

### Тема
- Светлая (стиль Notion/Figma) + тёмная. Переключатель в топбаре и в Tweaks.

### Палитра ступеней (1–7)
Каждая ступень — свой hue в OKLCH, одинаковый chroma и lightness. Делает гармонию читаемой и согласованной между компонентами.

| Ступень | Цвет (light) | Роль |
|---|---|---|
| 1 | красно-оранжевый | тоника |
| 2 | оранжевый | супертоника |
| 3 | оливково-жёлтый | медианта |
| 4 | зелёный | субдоминанта |
| 5 | синий | доминанта |
| 6 | фиолетовый | субмедианта |
| 7 | пурпурный | вводный |

Вторичные (вне тональности) — нейтральный фиолетовый `--secondary`, в круге — пунктир.

### Шрифты
- Inter — UI
- JetBrains Mono — ноты, аппликатуры, табличные числа

## Раскладка

Три колонки:
- **Левая (232px):** инструмент / строй / тональность / диатоника / вторичные доминанты
- **Центральная (flex):** активный инструмент, квинто-квартовый круг (сворачиваемый), полоса закреплённых
- **Правая (320px):** поисковик аккордов

### Адаптивность
- **Брейкпоинт `1024px`** (изначально 640, бампнут после фидбэка по тесному main): ниже — боковые становятся выезжающими (slide-in overlay), управляются `menu`/`search` кнопками в топбаре.
- **На широком экране** боковые скрываются вручную кнопками `sidebar-left` / `sidebar-right` в топбаре — грид перестраивается (две колонки или одна).
- Закреплённые аккорды — горизонтальная полоса внизу основной области.
- Tweaks-панель — правый нижний угол, только в Claude Design (по протоколу edit-mode).

## Функциональность

### Готово

#### Инструменты
- [x] Гитара 6 струн
- [x] Бас 5 струн
- [x] Фортепиано 3 октавы
- [x] Гитарные пресеты строёв: Standard, Drop D, DADGAD, Open G, Open D, Half-step down
- [x] Басовые пресеты: Standard, Drop A, Tenor
- [x] Каподастр (гитара) — 0–5, 7
- [x] Маркеры ладов 3, 5, 7, 9, 15 + двойной 12
- [x] Подсветка нот по ступени тональности (цветные точки 1–7)
- [x] При выборе аккорда — лейблы R / 2 / 3 / 5 / 7 на нотах аккорда

#### Тональность
- [x] Корень + лад (major / minor)
- [x] Top-bar key picker с дропдауном
- [x] Левая панель: чипы major/minor + 12 корней
- [x] Полная гамма в топбаре

#### Квинто-квартовый круг
- [x] Внешнее кольцо — мажоры, внутреннее — относительные миноры
- [x] Диатонические сектора — цветами своих ступеней
- [x] Вторичные доминанты — пунктиром
- [x] Клик по сектору меняет тональность
- [x] Сворачивается, рядом — мини-карточки диатонических аккордов

#### Поисковик аккордов
- [x] Три таба: По имени / По нотам / По позиции
- [x] Поиск по имени (startsWith + includes)
- [x] Подбор по выбранным нотам
- [x] Фильтры типа: maj / min / dim / sus
- [x] Фильтры расширений: 7 / 9 / 11
- [x] Карточки результатов с диаграммой (когда форма есть в библиотеке) или с нотами
- [x] Кнопка пина

#### Аккорды
- [x] ChordDiagram — нут / лады / струны / пальцы / open-mute индикаторы / детект баррэ
- [x] Библиотека из 20 базовых форм (C, G, D, A, E, F, Am, Em, Dm, Bm, Cmaj7, G7, ...)
- [x] Закреплённая полоса с горизонтальным скроллом
- [x] Клик по диатоническому/закреплённому/найденному аккорду — подсвечивает его на грифе
- [x] Кнопка `×` в чипе активного аккорда — снимает выделение

#### Tweaks
- [x] Тёмная тема
- [x] Подсветка: degree / note (UI готов; на инструменте поведение пока одинаковое)
- [x] Плотность: compact / regular
- [x] Римские в круге — toggle

#### UX-разное
- [x] Адаптивная раскладка (брейкпоинт 1024)
- [x] Ручное сворачивание боковых панелей кнопками в топбаре
- [x] Скрол-оверлей на узких viewports — клик по затемнённому фону закрывает обе панели

## Тех-стек

HTML + React 18 (UMD) + Babel Standalone. Без сборки, без npm. Любой статический сервер из корня:

```bash
python3 -m http.server 8765
```

Файлы — см. `CLAUDE.md`.

## История изменений

- **Initial export** из Claude Design — полный прототип со всеми фичами из секции "Готово" кроме ручного сворачивания.
- **Adaptive layout v2** — брейкпоинт 640 → 1024; добавлены кнопки `sidebar-left` / `sidebar-right` в топбаре для ручного сворачивания на широких экранах. Причина: пользователь использует тайлинг, и main-область чувствовалась тесно при двух открытых панелях.
